# Intelligent Study Assistant using AI & ML

## 📌 Overview
This project is an AI/ML-based system that predicts a student's focus level and provides study advice based on input conditions like sleep, study hours, time of day, and mood.

---

## 🎯 Problem Statement
Students often struggle with maintaining focus while studying. This project aims to predict whether a student will be focused and suggest appropriate actions.

---

## ⚙️ Features
- Predicts focus level (High / Low)
- Provides confidence score
- Gives personalized study advice
- Combines Machine Learning and Rule-Based AI

---

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Machine Learning (Logistic Regression)

---

## 📊 How it Works
1. Dataset is created manually
2. Model is trained using Logistic Regression
3. User inputs conditions
4. Model predicts focus
5. Rule-based system gives advice

---

## ▶️ How to Run

### Step 1: Train Model
```
python train_model.py
```

### Step 2: Run Prediction
```
python predict.py
```
---

## 📌 Example

Input:
- Sleep: 6  
- Study: 2  
- Time: afternoon  
- Mood: tired  

Output:
- Focus Prediction: Low  
- Confidence: 70%  
- Advice: Take a short break and revise easy topics
